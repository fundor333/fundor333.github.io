    SafeRoutes - applicazione per Android, vincitrice del 3 posto all' hackathon EODAppathon, organizzato dalla regione Veneto. L'app ti permette di scegliere un sentiero che vuoi percorrere e veni avvisato di eventuali rischi mentre percorri il sentiero. Utilizza gli Open Data della regione Veneto attraverso le api forniteci. Fa ampio uso di un database in real-time per la valutazione dei percorsi realizzato in FireBase
